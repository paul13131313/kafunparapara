# 花粉パラパラ（KAFUN PARAPARA）仕様書
> Claude Code 用スペックドキュメント

---

## コンセプト

**ギャルが花粉の発生源。**
ギャルがパラパラを踊ると、手の先から大量の花粉がパラパラと飛び散る。
花粉量が多い日ほど、ギャルは激しく踊り花粉を撒き散らす。
花粉が少ない日は地べたに座ってスマホ見たり、前髪いじってだるそうにしている。
花粉トラッキングサイトとしてリアルデータ連動。

- **リポジトリ名**: `kafunparapara`
- **開発ディレクトリ**: `naganohiroshi/dev/kafunparapara`
- **デプロイ先**: Vercel
- **技術スタック**: Next.js (App Router) + TypeScript + Tailwind CSS

## ① アセット要件（ユーザーが用意するもの）

### キャラクター動画（ループ映像）

5段階の花粉レベルに対応した**シームレスループ動画を5本**。
ギャルキャラがパラパラを踊り、その動きで花粉が舞う。

| ファイル名 | 花粉レベル | キャラクターの状態 |
|-----------|-----------|-----------------|
| `state-1.webm` | 少ない（〜29） | 地べた座り・スマホ見てる |
| `state-2.webm` | やや多い（30〜99） | 前髪いじりながらぼーっとしてる |
| `state-3.webm` | 多い（100〜299） | 立ち上がってゆっくり踊り始める |
| `state-4.webm` | 非常に多い（300〜） | フルパワーでパラパラ・手から花粉まき散らし |
| `state-5.webm` | 花粉爆発（600〜） | 超ハイテンション・手から花粉が폭発 |

> **ロジック**: ギャルが花粉の発生源。花粉が多い日ほどギャルは激しく踊り、手の先から花粉がパラパラ飛び散る。
> 花粉が少ない日はやることなくてだるそうにしている。

#### 花粉パーティクルについて

- キャラのダンス映像に花粉粒子（黄色い小さい粒）を含めて撮影・合成するか
- または映像はキャラのみにして、サイト側でCSS/JSパーティクルを重ねる（推奨）

#### 動画スペック

| 項目 | 仕様 |
|------|------|
| フォーマット | **WebM（VP9コーデック）＋アルファチャンネル（透過背景）** |
| 解像度 | **1080 × 1080px**（正方形） |
| フレームレート | **30fps** |
| ループ | **シームレスループ**（最終フレームと最初フレームが自然につながること） |
| 背景 | **完全透明**（透過あり） |
| ファイルサイズ目安 | 各5MB以内 |
| 秒数 | 3〜8秒程度 |

After Effectsの書き出し設定:
```
形式: WebM / チャンネル: RGB + アルファ / コーデック: VP9
```

## ④ 花粉パーティクル仕様（JS実装）

- `canvas`要素をキャラの上にフルスクリーンで重ねる
- **発生源はキャラの手の先**（画面中央やや下の左右）
- 黄色い小粒（直径2〜5px, `#FFD700`系）が手元から外側に飛び散る
- 花粉レベルが低い（state-1〜2）ときはパーティクルほぼなし or ゼロ
- 花粉レベルが上がるにつれ、手から勢いよくパーティクルが噴出
- 重力で下に落ちながらふわふわ漂う動き

```
レベル1: 0〜5粒・ほぼなし（ギャル座ってるので）
レベル2: 10粒・少量
レベル3: 80粒・手元から飛び始める
レベル4: 250粒・バンバン出る
レベル5: 600粒・爆発・画面埋め尽くす
```
